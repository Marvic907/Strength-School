# Branding assets placeholder (text-only)

This repository intentionally avoids committing binary image files so PR automation can run in environments that reject binary diffs.

## How to apply official Strength School Malta branding
Add these files from the official brand pack before release:
- `assets/branding/strength-school-logo.png` (in-app header and PDF)
- `assets/icon.png` (app launcher icon)
- `assets/adaptive-icon.png` (Android adaptive icon)
- `assets/splash.png` (splash image)
- `assets/favicon.png` (web favicon)

Then update `app.json` to point to those image paths.
